## Error States

### Missing Tool Selection (on connect attempt)
```
Scenario: User tries to connect edge from node with no tool selected

Behavior:
- Connection line appears dashed red
- Tooltip on source node: "Select a tool first"
- Edge creation blocked
- Source node dropdown pulses once (attention cue)
```

### Invalid Edge (self-loop)
```
Scenario: User tries to connect node to itself

Behavior:
- Connection snaps back (spring animation)
- Brief toast: "Can't connect a node to itself"
- No edge created
```

### Invalid Edge (duplicate)
```
Scenario: User tries to create edge that already exists

Behavior:
- Existing edge highlights (pulse)
- Toast: "These nodes are already connected"
- No duplicate created
```

### Unknown Tool in Registry
```
Scenario: Registry JSON references tool not found

Behavior:
- Node renders with warning badge
- Tool dropdown shows: "âš ï¸ Unknown tool"
- Vibe calculated as NEUTRAL for all edges
- Inspector shows: "Tool not found in registry"
```

### Corrupt localStorage
```
Scenario: localStorage graph data is malformed

Behavior:
- Catch parse error
- Log to console
- Clear corrupt data
- Load seeded graph
- Toast: "Couldn't load saved stack. Starting fresh."
```
